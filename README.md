# Morganite: Local Blossom Cache for Android

Morganite is a Local Blossom Cache for Android

# Current Features

- [x] Health check endpoint `HEAD /`
- [x] Blob Retrieval `GET` and `HEAD`
- [x] Range Requests
- [x] Proxy Hints `xs=<server>` and `as=<pubkey>`
- [x] LRU Cache (When the app reaches 1GB it prunes the least used blobs until 850MB)

More on how this works in the [Blossom repo](https://github.com/hzrd149/blossom/blob/master/implementations/local-blossom-cache.md)
